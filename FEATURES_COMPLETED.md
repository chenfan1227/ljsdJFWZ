## 🔧 **使用方法**

### **功能入口**
1. **编辑资料**: 个人中心 → "✏️ 编辑" 按钮
2. **账户设置**: 个人中心 → "⚙️ 设置" 按钮
3. **积分历史**: 通过 `showPointsHistory()` 函数调用
4. **提现申请**: 通过 `showWithdrawalRequest()` 函数调用

### **集成方式**
```html
<!-- 在app-demo.html中引入 -->
<script src="user-profile-functions.js"></script>
```

### **API调用示例**
```javascript
// 显示编辑资料
editProfile(); // 替换原有的alert调用

// 显示设置
showSettings(); // 替换原有的alert调用

// 显示积分历史
showPointsHistory();

// 显示提现申请
showWithdrawalRequest();
```
