```bash

```

### 简单启动（仅前端）
```bash
# 启动前端演示
./start.sh
```

### 手动启动
```bash
# 1. 设置数据库
cd backend && node scripts/setup-database.js

# 2. 启动后端
npm start &

# 3. 启动前端
cd .. && python3 -m http.server 8080
```
